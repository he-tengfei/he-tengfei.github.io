常见报错

发表于2022-10-26|更新于2022-10-26
|字数总计:312|阅读时长:1分钟|阅读量:

Java相关

错误1：

报错：找不到文件

解决：源文件名不存在或写错，或者当前路径错误。如果是包找不到，不要把scope设为provided

错误2：

报错：主类名和文件名不一致

解决：声明为public的主类应与文件名一致，否则编译失败

MySQL相关

错误1：

报错：is not allowed to connect to this MySQL server

解决：原因是本地MySQL不允许远程登录，把这里改成%即可。

Git

错误1：

报错：Failed to connect to github.com port 443 after 21132 ms: Timed out

或者：OpenSSL SSL_read: Connection was reset, errno 10054

解决：网不好，多试几次

Hive

报错：return code 3 from org.apache.hadoop.hive.ql.exec.spark.SparkTask

这种就直接去http://hadoop180:9870/spark-histor里看日志，搜due to或者Caused by，比如这次的报错信息，意思就是CastLongToString函数找不到，这是spark引擎和hive的版本冲突问题，把cast函数去掉就可以了。

Python

错误1：

报错：Unicode-objects must be encoded before hashing

解决：hashlib.sha256(data)函数中，data参数的类型应该是bytes，hash前必须把数据转换成bytes类型
