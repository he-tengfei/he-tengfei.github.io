1. еҲҶеҢәдёҺеҲҶз»„

1.1 MapReduceдёӯж•°жҚ®жөҒеҠЁ

  • жңҖз®ҖеҚ•зҡ„иҝҮзЁӢпјҡmap -> reduce
  • е®ҡеҲ¶дәҶpartitionerд»Ҙе°Ҷmapзҡ„з»“жһңйҖҒеҫҖжҢҮе®ҡreducerзҡ„иҝҮзЁӢпјҡmap -> partition -> reduce
  • еўһеҠ дәҶеңЁжң¬ең°е…ҲиҝӣжҖ§дёҖж¬Ўreduce(дјҳеҢ–)иҝҮзЁӢпјҡmap -> combine(жң¬ең°reduce) -> partition -> reduce

1.2 Partitionзҡ„еҺҹзҗҶе’ҢдҪңз”Ё

еҫ—еҲ°mapз»ҷзҡ„и®°еҪ•еҗҺпјҢиҜҘеҲҶй…Қз»ҷе“ӘдәӣreducerжқҘеӨ„зҗҶе‘ўпјҹ

hadoopй»ҳи®Өзҡ„HashPartitionerе№¶дёҚдёҖе®ҡиғҪж»Ўи¶іжҲ‘д»¬зҡ„йңҖжұӮгҖӮдҫӢеҰӮпјҢз»ҸиҝҮpartitionеӨ„зҗҶеҗҺпјҢдёҖдёӘиҠӮзӮ№зҡ„reducerеҲҶй…ҚеҲ°дәҶ20жқЎи®°еҪ•пјҢеҸҰдёҖдёӘеҚҙеҲҶй…ҚйҒ“дәҶ10WдёҮжқЎпјҢиҜ•жғіпјҢиҝҷз§Қжғ…еҶөж•ҲзҺҮеҰӮдҪ•гҖӮеҸҲжҲ–иҖ…пјҢжҲ‘д»¬жғіиҰҒеӨ„зҗҶеҗҺеҫ—еҲ°зҡ„ж–Үд»¶жҢүз…§дёҖе®ҡзҡ„и§„еҫӢиҝӣиЎҢиҫ“еҮәпјҢиҝҷдәӣй»ҳи®Өзҡ„partitionerжҳҜеҒҡдёҚеҲ°зҡ„гҖӮ

жүҖд»ҘйңҖиҰҒжҲ‘д»¬иҮӘе·ұе®ҡеҲ¶partitionжқҘж №жҚ®иҮӘе·ұзҡ„иҰҒжұӮпјҢйҖүжӢ©и®°еҪ•зҡ„reducerгҖӮиҮӘе®ҡд№үpartitionerеҫҲз®ҖеҚ•пјҢеҸӘиҰҒиҮӘе®ҡд№үдёҖдёӘзұ»пјҢе№¶дё”з»§жүҝPartitionerзұ»пјҢйҮҚеҶҷе…¶getPartitionж–№жі•е°ұеҘҪдәҶпјҢеңЁдҪҝз”Ёзҡ„ж—¶еҖҷйҖҡиҝҮи°ғз”ЁJobзҡ„setPartitionerClassжҢҮе®ҡдёҖдёӢеҚіеҸҜгҖӮ

Mapзҡ„з»“жһңпјҢдјҡйҖҡиҝҮpartitionеҲҶеҸ‘еҲ°ReducerдёҠгҖӮMapperзҡ„з»“жһңпјҢеҸҜиғҪйҖҒеҲ°CombinerеҒҡеҗҲе№¶пјҢCombinerеңЁзі»з»ҹдёӯе№¶жІЎжңүиҮӘе·ұзҡ„еҹәзұ»пјҢиҖҢжҳҜз”ЁReducerдҪңдёәCombinerзҡ„еҹәзұ»пјҢд»–д»¬еҜ№еӨ–зҡ„еҠҹиғҪжҳҜдёҖж ·зҡ„пјҢеҸӘжҳҜдҪҝз”Ёзҡ„дҪҚзҪ®е’ҢдҪҝз”Ёж—¶зҡ„дёҠдёӢж–ҮдёҚеӨӘдёҖж ·иҖҢе·ІгҖӮMapperжңҖз»ҲеӨ„зҗҶзҡ„й”®еҖјеҜ№<key,value>пјҢжҳҜйңҖиҰҒйҖҒеҲ°ReducerеҺ»еҗҲе№¶зҡ„пјҢеҗҲе№¶зҡ„ж—¶еҖҷпјҢжңүзӣёеҗҢkeyзҡ„й”®/еҖјеҜ№дјҡйҖҒеҲ°еҗҢдёҖдёӘReducerйӮЈгҖӮ

е“ӘдёӘkeyеҲ°е“ӘдёӘReducerзҡ„еҲҶй…ҚиҝҮзЁӢпјҢжҳҜз”ұPartitionerи§„е®ҡзҡ„гҖӮе®ғеҸӘжңүдёҖдёӘж–№жі•пјҢ**getPartition(Textkey,Textvalue,intnumPartitions)**гҖӮиҫ“е…ҘжҳҜMapзҡ„з»“жһңеҜ№<key,value>е’ҢReducerзҡ„ж•°зӣ®пјҢиҫ“еҮәеҲҷжҳҜеҲҶй…Қзҡ„Reducer(ж•ҙж•°зј–еҸ·)гҖӮе°ұжҳҜжҢҮе®ҡMapprиҫ“еҮәзҡ„й”®еҖјеҜ№еҲ°е“ӘдёҖдёӘreducerдёҠеҺ»гҖӮзі»з»ҹзјәзңҒзҡ„PartitionerжҳҜHashPartitionerпјҢе®ғд»Ҙkeyзҡ„HashеҖјеҜ№Reducerзҡ„ж•°зӣ®еҸ–жЁЎпјҢеҫ—еҲ°еҜ№еә”зҡ„ReducerгҖӮиҝҷж ·дҝқиҜҒеҰӮжһңжңүзӣёеҗҢзҡ„keyеҖјпјҢиӮҜе®ҡиў«еҲҶй…ҚеҲ°еҗҢдёҖдёӘreducreдёҠгҖӮеҰӮжһңжңүNдёӘreducerпјҢзј–еҸ·е°ұ0,1,2,3вҖҰвҖҰ(N -> 1)гҖӮ

1.3 Partitionзҡ„дҪҝз”Ё

еҲҶеҢәеҮәзҺ°зҡ„еҝ…иҰҒжҖ§пјҢжҜ”еҰӮеҰӮдҪ•дҪҝз”ЁHadoopдә§з”ҹдёҖдёӘе…ЁеұҖжҺ’еәҸзҡ„ж–Үд»¶пјҹ

жңҖз®ҖеҚ•зҡ„ж–№жі•е°ұжҳҜдҪҝз”ЁдёҖдёӘеҲҶеҢәпјҢдҪҶжҳҜиҜҘж–№жі•еңЁеӨ„зҗҶеӨ§еһӢж–Үд»¶ж—¶ж•ҲзҺҮжһҒдҪҺпјҢеӣ дёәдёҖеҸ°жңәеҷЁеҝ…йЎ»еӨ„зҗҶжүҖжңүиҫ“еҮәж–Үд»¶пјҢд»ҺиҖҢе®Ңе…Ёдё§еӨұдәҶMapReduceжүҖжҸҗдҫӣзҡ„е№¶иЎҢжһ¶жһ„зҡ„дјҳеҠҝгҖӮ

дәӢе®һдёҠжҲ‘д»¬еҸҜд»Ҙиҝҷж ·еҒҡпјҢйҰ–е…ҲеҲӣе»әдёҖзі»еҲ—жҺ’еҘҪеәҸзҡ„ж–Үд»¶пјӣе…¶ж¬ЎпјҢдёІиҒ”иҝҷдәӣж–Үд»¶(зұ»дјјдәҺеҪ’е№¶жҺ’еәҸ)пјӣжңҖеҗҺеҫ—еҲ°дёҖдёӘе…ЁеұҖжңүеәҸзҡ„ж–Үд»¶гҖӮдё»иҰҒзҡ„жҖқи·ҜжҳҜдҪҝз”ЁдёҖдёӘpartitionerжқҘжҸҸиҝ°е…ЁеұҖжҺ’еәҸзҡ„иҫ“еҮәгҖӮжҜ”ж–№иҜҙжҲ‘д»¬жңү1000дёӘ1 -> 10000зҡ„ж•°жҚ®пјҢи·‘10дёӘruduceд»»еҠЎпјҢеҰӮжһңжҲ‘д»¬иҝҗиЎҢиҝӣиЎҢpartitionзҡ„ж—¶еҖҷпјҢиғҪеӨҹе°ҶеңЁ1 -> 1000дёӯж•°жҚ®зҡ„еҲҶй…ҚеҲ°з¬¬дёҖдёӘreduceдёӯпјҢ1001 -> 2000зҡ„ж•°жҚ®еҲҶй…ҚеҲ°з¬¬дәҢдёӘreduceдёӯпјҢд»ҘжӯӨзұ»жҺЁгҖӮиҖҢеҲҶеҢәеҶ…жҳҜжңүеәҸзҡ„пјҢжүҖд»ҘжҲ‘д»¬еҸӘиҰҒcatжүҖжңүзҡ„иҫ“еҮәж–Үд»¶пјҢеҸҳжҲҗдёҖдёӘеӨ§зҡ„ж–Үд»¶пјҢе°ұйғҪжҳҜжңүеәҸзҡ„дәҶгҖӮ(еҰӮдҪ•дәҶи§Јж•°жҚ®зҡ„еҲҶеёғеҸҜд»Ҙз”ЁйҮҮж ·пјҢйҮҮж ·зҡ„е…·дҪ“дҪҝз”ЁеҸҜд»ҘзҷҫеәҰ)

1.4 еҲҶз»„зҡ„жҰӮеҝө

еҲҶеҢәжҳҜж №жҚ®keyжқҘеҶіе®ҡе“Әдәӣkey-valueиў«еҲҶеҲ°еҗҢдёҖдёӘreduceеӨ„зҗҶпјҢиҖҢеҲҶз»„жҳҜж №жҚ®keyжқҘеҶіе®ҡеҗҢдёҖдёӘreduceдёӯзҡ„key-valueеңЁеҗҢдёҖжү№ж¬ЎдёӯиҝӣиЎҢеӨ„зҗҶгҖӮеңЁreduceдёӯпјҢдёҖдёӘkeyеҜ№еә”зқҖдёҖдёӘиҝӯд»ЈеҷЁпјҢиҖҢиҝҷдәӣиҝӯд»ЈеҷЁдёӯзҡ„еҖје°ұжқҘиҮӘдәҺеҗҢдёҖз»„гҖӮ

жҜҸдёӘеҲҶеҢәжҳ е°„еҲ°дёҖдёӘreducerпјҢжҜҸдёӘеҲҶеҢәеҶ…еҸҲи°ғз”Ёjob.setSortComparatorClass(жҜ”иҫғеҷЁ)и®ҫзҪ®зҡ„keyжҜ”иҫғеҮҪж•°зұ»жҺ’еәҸгҖӮз„¶еҗҺејҖе§Ӣжһ„йҖ дёҖдёӘkeyеҜ№еә”зҡ„valueиҝӯд»ЈеҷЁгҖӮиҝҷж—¶е°ұиҰҒз”ЁеҲ°еҲҶз»„пјҢдҪҝз”Ёjob.setGroupingComparatorClass(еҲҶз»„жҜ”иҫғеҷЁ)и®ҫзҪ®зҡ„еҲҶз»„еҮҪж•°зұ»гҖӮеҸӘиҰҒиҝҷдёӘжҜ”иҫғеҷЁжҜ”иҫғзҡ„дёӨдёӘkeyзӣёеҗҢпјҢд»–д»¬е°ұеұһдәҺеҗҢдёҖдёӘз»„(е®ғд»¬зҡ„valueж”ҫеңЁдёҖдёӘvalueиҝӯд»ЈеҷЁпјҢиҖҢиҝҷдёӘиҝӯд»ЈеҷЁзҡ„keyдҪҝз”ЁеұһдәҺеҗҢдёҖдёӘз»„зҡ„жүҖжңүkeyзҡ„з¬¬дёҖдёӘkeyгҖӮ
