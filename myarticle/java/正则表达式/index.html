正则总的来说就三个词：分组捕获匹配

老韩对这块儿知识的评价：非常实用！！！

一、快速入门

1. 为什么要学习正则表达式？

因为极其方便！！！体验正则表达式的威力：

  • 提取文章中所有的英文单词
  • 提取文章中所有的数字
  • 提取文章中所有的英文单词和数字
  • 提取百度热榜 标题
  • 匹配ip地址

再提出几个问题：

  • 给你一个字符串，如何找出所有4个数字连在一起的字符？
  • 不仅要满足上个条件，还要让第一位和第四位相同，第二位和第三位相同，比如1221，这种又该如何实现？
  • 验证输入的字符串是否符合电子邮件格式？
  • 验证输入的手机号是否符合手机号格式？

解决之道 -> 正则表达式：

  • Java为了解决上述问题专门提供了正则表达式技术，专门用于处理类似的文本问题
  • 简单的说：正则表达式就是对字符串执行模式匹配的技术
  • 正则表达式：regular expression -> RegExp

2. RegExp基本介绍

一个正则表达式，就是用某种模式去匹配字符串的一个公式。公式看上去古怪且复杂，但是一旦清楚它的原理和规则后，写起来相当简单，而且对文本处理工作效率的提升肉眼可见。并且正则表达式并不是只有Java才有，很多编程语言都支持正则。

三、正则表达式语法

1. 基本介绍

要想灵活的运用正则表达式，必须了解其中各种元字符的功能，元字符从功能上大致分为6种：

  • 限定符，限定出现的次数
  • 选择匹配符，|，相当于or
  • 分组组合和反向引用符
  • 特殊字符
  • 字符匹配符
  • 定位符
  • 注意不要随便加空格，会进入匹配规则，除非你就想匹配到空格

2. 元字符 - 转义符 \\

特殊字符需要使用转义符，否则检索不到结果，甚至会报错

需要使用转义符的15个特殊字符：. * + ( ) $ / \ ? [ ] ^ { } -

3. 元字符 - 字符匹配符

符号 含义 匹配输入
[ ] 可接收的单个字符列表，里面的字符都是真实字符
[^] 不接收的单个字符列表
- 连字符，用来表示范围
. 匹配除\n(换行)之外的任何单个字符，如果要单独匹配.本身，则要使用\\.
\\d 匹配单个数字字符，相当于[0-9]
\\D 匹配单个非数字字符，相当于[ ^0-9 ]
\\w 匹配单个数字、大小写字母和下划线，相当于[0-9a-zA-Z_]
\\W 匹配单个非数字、大小写字母和下划线，相当于[ ^0-9a-zA-Z_ ]
\\s 匹配任何空白字符，比如空格、制表符等
\\S 匹配任何非空白字符，和\\s刚好相反
[efgh] 接收e、f、g、h中的任意单个字符
[^abc] 接收除a、b、c之外的任意单个个字符，包括数字和特殊符号
A-Z 表示范围，接收A到Z之间的任意单个大写字母
a..b 以a开头，b结尾的任意长度为4的字符串 a35b
\\d{3}(\\d)? 包含3个或4个数字的字符串，后面的?代表有1个或者0个 123
\\D(\\d)* 以单个非数字字符开头，后接任意个数字字符串 a12
\\W+\\d{2} 以至少1个非数字字母字符下划线开头，后面跟2个数字的字符串 #?@10

4. 元字符 - 选择匹配符 |

相当于 or

5. 元字符 - 限定符

  • java匹配默认贪婪匹配，即尽可能匹配多的

注意下面圈起来的，没有括号，?就只作用于c，{n}改为匹配n次更好

符号 含义 匹配输入
* 重复0次或多次
+ 重复1次或多次
? 重复0次或1次
{n} 重复n次
{n,} 至少重复n次
{n,m} 至少重复n次，但不多于m次
m+{abc}* m、mmabc、mabcabc
m+abc? 没有括号的话，?只对c生效 mab、mmabc
[abcd]{3,} 长度至少为3 aaa、abcd

6. 元字符 - 定位符

规定要匹配的字符串出现的位置，一般定位符都是对整个字符串进行匹配，而不是字符串的子串

符号 含义 匹配输入
^ 匹配开头
$ 匹配结尾
\\b 匹配空格隔开的后边界，border
\\B 匹配前边界
^[0-9]\\-[a-z]+$ 以1个数字开头后接连字符，并以至少1个小写字母结尾 1-a
^$ 匹配空行

7. 捕获分组

命名分组的命名规则：(?pattern)，name不能含特殊符号，不能以数字开头，尖括号可替换为单引号。

10. 分组、捕获、反向引用

问题引出：

给你一段文本，请你找出所有4个数字连在一起的字符串，并且这4个数字要满足：第1位与第4位相同，第2位与第3位相同，比如1221

要想解决上面的问题，必须了解正则表达式的几个概念：

  • 分组：一对圆括号代表一个分组
  • 捕获：把正则表达式匹配到的内容，保存到内存中以数字编号或显式命名的组里，方便后面调用，注意组0代表整个匹配到的内容
  • 反向引用：圆括号的内容被捕获后，可以在这个括号后被使用，从而写出一个比较实用的匹配模式，这个我们称为反向引用。反向引用即可以在正则表达式内部，也可以在外部，内部反向引用方式为：\\分组号，外部反向引用方式为：$分组号

五、正则的3个常用类

java.util.regex包主要包括以下三个类：Pattern类、Matcher类和PatternSyntaxException类

  • Pattern类：Pattern类没有公共构造方法，创建一个Pattern对象需要调用它的公共静态方法，它返回一个pattern对象，该方法接受一个正则表达式作为它的第一个参数，比如：Pattern pattern = Pattern.compile(regStr);
  • Matcher类：Matcher对象是对输入字符串进行解释和匹配的引擎，与Pattern类一样，Matcher类也没有公共构造方法，需要调用Pattern对象的matcher方法来获得一个Matcher对象，Matcher类常用的方法有start()，end()，matches()，replaceAll()
  • PatternSyntaxException类：是一个非强制异常类，它表示一个正则表达式中的语法错误
